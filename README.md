# 🌍 Global Tax Calculator

A premium, interactive web application for real-time global salary breakdowns, tax analytics, and professional PDF reporting.

## ✨ Key Features

* **Multi-Regional Tax Engines**: Support for USA (Federal 2025), UK (2024-2025), India (FY 2025-26 New Regime), and Canada (Federal + Ontario 2025).
* **Smart Logic Implementation**: Includes high-earner allowance tapering (UK), Section 87A tax rebates (India), and combined federal/provincial brackets (Canada).
* **Interactive Analytics**: Dynamic doughnut charts visualizing Net Income vs. Total Tax using Chart.js.
* **Custom Deduction Manager**: Add and remove multiple custom allowances or deductions in real-time.
* **PDF Export**: Generate structured, band-by-band tax reports for financial planning.
* **Premium UI**: Fully responsive glassmorphism design built for desktop and mobile.

## 🛠️ Technical Stack

* **Runtime/Bundler**: Vite.
* **Language**: TypeScript / JavaScript (ES6+).
* **Visualization**: Chart.js.
* **Document Generation**: jsPDF.
* **Styling**: CSS3 with CSS Variables and Backdrop Filters.

## 🚀 Getting Started

### Prerequisites
* Node.js (v18.0.0 or higher).
* npm or yarn.

### Installation
1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/atharvapisal16/global-tax-calculator.git](https://github.com/atharvapisal16/global-tax-calculator.git)
    cd Global-Tax-Calculator
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```

### Development
